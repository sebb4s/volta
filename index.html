PREGUNTAS FRECUENTES

Todo lo que necesitás saber para empezar

¿Se requieren conocimientos previos para los cursos?

No! Todos los cursos empiezan desde cero, así que no hacen falta conocimientos previos. Únicamente el curso de AUTOMATIZACIÓN INDUSTRIAL que se recomiendan conocimientos del curso de ELECTRICIDAD DOMICILIARIA.

¿Cómo se realizan los pagos?

Se abona la primer cuota para empezar y las siguientes cuotas deben pagarse del 1er al 10mo día de cada mes.

¿Qué métodos de pago aceptan?

Como medio de pago aceptamos efectivo, mercado pago y transferencia bancaria.

¿Van a haber prácticas de todo lo visto?

En absolutamente TODOS los cursos hay suficientes elementos para poder prácticar todo lo visto, así que sí, cada clase se compone tanto de teoría como de práctica.

¿Cuáles son las fechas de comienzo?

Envianos un mensaje a nuestro número (+54 9 11 6201-3619) y les estaremos mandando las próximas fechas de comienzo para el curso que a usted le interese.

¿Cómo es el proceso para obtener el certificado?

Una vez llegado el último tema del temario, se tomará un exámen evaluando lo visto durante el curso, si el mismo se encuentra aprobado, se le dará el certificado, de no ser así tendrá otra oportunidad con recuperatorio.

¿Te interesa algún curso?

Envianos un mensaje si estás interesado y te pasamos toda la información que necesites!
